# Simple-blog-app

Application name: Mysphere

Mysphere is a modern UI-focused social blog platform practice project developed to improve frontend development skills.

The project was built entirely with HTML, CSS, and Vanilla JavaScript.
Its main purpose is to gain experience in frontend concepts such as dynamic user interfaces, DOM manipulation, responsive design, and user interactions.

Features
User registration system
User login system
Data storage with LocalStorage
Dynamic page transitions
Responsive design
Modern dark theme interface
Profile page
Mail sending interface
Sent mail viewing system
Sidebar navigation structure
Hover and transition effects
SPA-like structure built with Vanilla JavaScript

Project Purpose

This project was developed to practice and improve skills in the following areas:

DOM manipulation
Dynamic element creation
Event handling
Responsive design
Modern UI development
LocalStorage usage
Page management logic
Component-based thinking

Development Notes

This project is a learning and practice project.

All data is stored in the browser using localStorage.
There is currently no real backend, database, or secure authentication system.


Technologies Used
HTML5
CSS3
Vanilla JavaScript
LocalStorage API
Font Awesome
